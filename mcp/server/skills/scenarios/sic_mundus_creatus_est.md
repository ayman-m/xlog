# Skill: Sic Mundus Creatus Est Scenario

## Category
scenarios

## Attack Type
Persistence / Defense Evasion

## MITRE ATT&CK Tactics
- TA0003: Persistence
- TA0005: Defense Evasion
- TA0002: Execution

## High-Level Guidance

Keep this scenario minimal:

1. Establish one new scheduled task or service on a single host.
2. Show a brief disablement or modification of a security control.
3. Include a quiet, low-frequency beacon to a single external domain.
4. End with a successful persistence check (task/service runs again).
