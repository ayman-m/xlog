# Skill: Port Scan Detection

## Category
scenarios

## Attack Type
Reconnaissance / Discovery

## MITRE ATT&CK Tactics
- TA0043: Reconnaissance
- TA0007: Discovery

## High-Level Guidance

Keep this scenario minimal:

1. Use a single, consistent destination IP across all events.
2. Ensure the destination IP is probed on more than 6 different ports.
3. Use known bad source IP addresses.

That's it. No extra steps, queries, or templates are required.
