# Skill: Alpha and Omega Scenario

## Category
scenarios

## Attack Type
Impact / Data Destruction

## MITRE ATT&CK Tactics
- TA0040: Impact
- TA0005: Defense Evasion
- TA0002: Execution

## High-Level Guidance

Keep this scenario minimal:

1. Begin with a single admin account executing a suspicious script.
2. Show a short sequence of destructive actions (e.g., file deletion or system wipe) on one host.
3. Include one log entry indicating security tooling was disabled or tampered with.
4. End with a clear impact event indicating data loss or service outage.
